# User Story

## Title
As a user, I want to browse and search for second-hand items so that I can find affordable products.

## Description
The system should allow users to view all available second-hand items, search items by category or keywords, and view item details.

## Acceptance Criteria
- User can view all items
- User can search items by category
- User can view item details by ID
- API returns proper responses (200, 404)

## Priority
High
